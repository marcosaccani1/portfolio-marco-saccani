Portfolio Web Personale - Marco Saccani

Descrizione del progetto
Questo progetto consiste nella realizzazione di un sito web personale sviluppato in HTML, CSS, Bootstrap e Sass.

L’obiettivo del sito è presentare il mio profilo professionale, le mie competenze tecniche, i miei progetti e un modo semplice per essere contattato.

Il sito include:
- Home page introduttiva
- Pagina Portfolio con progetti selezionati
- Pagina CV interamente realizzata in HTML
- Pagina Contatti con form

Tecnologie utilizzate
- HTML5
- CSS3
- Bootstrap 5
- Sass
- GitHub
- GitHub Pages
